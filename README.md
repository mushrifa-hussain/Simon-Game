# Simon Game 🎮

A fun memory game built using **HTML, CSS, JavaScript, and jQuery**.
Test your memory by remembering and repeating the color pattern! 🧠✨

## How the Game Works 🎯

* Press any key to start the game.
* The game flashes a random color.
* Click the same color to continue.
* Each level adds a new color to the sequence.
* Repeat the full pattern in the correct order.
* If you click the wrong color, the game is over ❌

## Features ⭐

* Level-by-level difficulty 📈
* Sound and button animations 🔊
* Game over effect ⚠️
* Restart and play again option 🔁
* Simple and interactive design 🎨
